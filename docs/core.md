# sillonCore

The core package is a collection of tools for the tool chain that are neither shared lirbary function (common) or apis (api). It contains for now the server to log run, and the engine to make each interface and command work. 

## Contains:
- Engine -> All the logic for the interface
- VersionControl -> Version control lirbary
- Simulation -> Logic of handling simulation object by the server
- glob -> IO for the glob object
- Envhandler -> IO for the .sillon
- sillonCLI -> [cli](simplycli)
- [Server](server)
- sillonGUI -> gui
